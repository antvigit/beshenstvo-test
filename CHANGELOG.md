# Changelog

Все заметные изменения в проекте задокументированы здесь.

## [v2.7] — 2026-06-26
### Добавлено
- В финальное сообщение добавлена ссылка на отчёт в GitHub Actions.
- Обновлены README и CHANGELOG, добавлен план по улучшению проекта.

### Исправлено
- Восстановлена работа скриншотов Allure-отчёта (возврат к HTTP-серверу с увеличенными таймаутами).
- Добавлена проверка готовности сервера перед созданием скриншота.

## [v2.6] — 2026-06-25
### Изменено
- **Возврат к стабильной конфигурации:** бот снова работает на Flask + polling (без Gunicorn).
- Удалены эксперименты с вебхуком и Gunicorn.
- Восстановлена стабильная отправка скриншотов и ссылок в Telegram.

## [v2.5] — 2026-06-25
### Изменено
- **Отказ от Gunicorn:** бот снова работает на встроенном Flask-сервере с `app.run()`.
- **Режим работы:** переключились на **polling** с параллельным health check для Render.

## [v2.4] — 2026-06-25
### Изменено
- **Production‑сервер для Telegram‑бота:** встроенный Flask‑сервер заменён на **Gunicorn** (впоследствии откачено).

## [v2.3] — 2026-06-25
### Добавлено
- **Контейнеризация тестов:** добавлен `Dockerfile` для запуска тестов в контейнере.
- **Полное логирование:** все `System.out.println` заменены на Log4j.
- **Параллельный запуск тестов внутри одного браузера:** включён параллелизм JUnit 5.

## [v2.2] — 2026-06-25
### Добавлено
- Health check эндпоинт `/health` для Render (бот не засыпает на бесплатном тарифе).

## [v2.1] — 2026-06-25

- Поддержка параллельного запуска тестов в двух браузерах через матрицу в GitHub Actions.
- Отдельные Allure-отчёты и скриншоты для каждого браузера.
- Бот переведён на вебхук (Flask) для стабильной работы (устранена ошибка 409).
- Отправка отчётов в чат пользователя, запустившего тесты.

### Исправлено
- Ошибка `tab crashed` в Chrome при работе через Grid.
- Предупреждения Node.js 20 в GitHub Actions.

## [v2.0] — 2026-06-24
### Добавлено
